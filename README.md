# NotesArtifacts

Keep this folder simple for future learning notes.

## Minimal structure

NotesArtifacts/
  notes/            # all your learning notes
  resources/        # images, screenshots, code snippets
  archive/          # old or completed notes

## Simple rule

- Put every new learning note in notes/
- Put supporting files in resources/
- Move old notes to archive/ when you no longer need them

## Best format

- Use HTML for visual notes and maps
- Use Markdown or plain text if you want faster note taking
